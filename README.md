# Rust学习

### 目录结构

```
learn_rust  //根目录
    |
    |- src  //源码
    |   |
    |   |- exercise //练习程序模块
    |   |- study    //rust学习内容
    |
    |- tests    //测试程序目录
```

### 使用

运行 `cargo test xxx` xxx是需要运行的测试模块或方法名称。

例如： `cargo test test_two_sum`
